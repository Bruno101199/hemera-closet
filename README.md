## Páginas

| Arquivo | Descrição |
|---|---|
| `index.html` | Página principal — hero, produtos, benefícios, mapa e rodapé |
| `produto.html` | Página de produto — galeria, seletor de cor/tamanho, página de vendas |

## Tecnologias

- HTML5 + CSS3 + JavaScript puro (sem frameworks)
- Google Fonts: Bebas Neue, Outfit, Cormorant Garamond
- Google Maps embed
